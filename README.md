# RechtFlott

Ein kleines, lokales 10-Finger-Training als statische Website mit zwei Modi.

## Start

- Doppelklick auf index.html
- Oder einen lokalen Server starten (wenn gewuenscht)
- Terminal: `./scripts/serve.ps1`
- Stoppen: `./scripts/stop.ps1`
